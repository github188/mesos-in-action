Mesos实战
=======

Apache Mesos是Apache软件基金会下属的顶级开源项目，它是目前云计算开源领域关于分布式计算方面最具生产价值的项目。在它出师不久就已经在Twitter、AirBnb、Apple、Mesosphere等公司大显身手，目前被全球关注云计算的公司作为重点应用的对象之一。
